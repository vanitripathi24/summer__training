# Day 1 - Git and GitHub Basics

## Topics Learned

- Introduction to Git
- Introduction to GitHub
- Creating a Repository
- Cloning a Repository
- Git Commands
- Staging Changes
- Committing Changes
- Pushing Code to GitHub

## Commands Practiced

git init
git add .
git commit -m "Initial Commit"
git push origin main

## Learning Outcome

Today I learned how version control works using Git and how GitHub helps in managing and sharing code repositories. I also learned how to create repositories, track changes, commit updates, and push code to GitHub.
